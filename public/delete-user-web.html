🗑️ Delete User via URL

Test the web-based user deletion using GET method with URL parameters

⚠️ Warning: Permanent Action

This feature allows you to delete users by visiting a URL. This action is permanent and cannot be undone.

  • User account will be permanently deleted
  • All user tokens will be invalidated
  • Admin users cannot be deleted via this method
  • You must be logged in as admin to use this feature
ℹ️ How It Works

Simply provide a user ID in the URL parameter, and the system will delete that user from the database. The result will be displayed on a dedicated page.

📖 URL Format & Examples

The URL follows this format:

http://localhost/admin/delete-user?user_id=USER_ID

Examples:

http://localhost/admin/delete-user?user_id=1
http://localhost/admin/delete-user?user_id=42
http://localhost/admin/delete-user?user_id=999
